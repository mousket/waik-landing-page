# Task 5c1-07 — A6 High-risk Residents (Top 5) Card + “Why now”
## Phase: 5c-1 — Admin/DON Daily Command
## Estimated Time: 3–5 hours
## Depends On: 5c1-10 data contract plan (risk signals)

---

## Why This Task Exists

Daily Command should not be purely reactive. The DON needs a small, prioritized list of residents who need proactive attention **today**.

---

## Card Contents (A6)

- Show **top 5** residents max.
- Each row includes:
  - resident name + room (+ unit if available)
  - 2–3 driver pills (e.g., repeat falls, anticoagulant, new confusion, behavior escalation)
  - a short “why now” sentence (1 line)
  - single CTA: “Open risk bundle”

---

## Guardrails

- No opaque “risk score” without explanation.
- Always show the drivers; avoid scary language.

---

## Success Criteria

- [ ] Card renders in A6 position.
- [ ] Never shows more than 5 rows.
- [ ] Every row has drivers + one CTA.
