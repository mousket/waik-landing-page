# Task 5c2-08 — E7 Intervention Effectiveness (Before/After Snapshots)
## Phase: 5c-2 — Admin/DON Executive View (Trends)
## Estimated Time: 3–6 hours
## Depends On: 5c2-11 data plan (interventions + events)

---

## Why This Task Exists

Leaders need to know whether interventions and process changes are *helping*.
This should be lightweight and honest — no overclaiming.

---

## Card Contents (E7)

Show up to **3 “what we tried” items** (configurable later):

Each item includes:
- Intervention label (e.g., “Hourly rounding implemented in Memory Care”)
- Before vs after snapshot:
  - incidents type count
  - repeat count
  - documentation completion
- Confidence framing:
  - “Directional signal” (not causal)
- CTA: “View evidence”

---

## Guardrails

- No causal language (“caused”, “reduced because of”).
- Must link to the before/after evidence sets.

---

## Success Criteria

- [ ] Max 3 items shown.
- [ ] Before/after shows the same metric definitions consistently.
- [ ] Evidence drilldowns work.
