# Task 5c2-06 — E5 Pattern & Cluster Insights (Evidence-backed)
## Phase: 5c-2 — Admin/DON Executive View (Trends)
## Estimated Time: 4–7 hours
## Depends On: 5c2-11 data plan (events + timestamps + unit/shift metadata)

---

## Why This Task Exists

The executive view’s superpower is pattern recognition. But it must be **trustworthy**: insights must be explainable and cite underlying records.

---

## Card Contents (E5)

Show up to **3 insights max** per range (never more):

Insight template:
- **Title**: “Falls clustering during handoff window”
- **Evidence**: “6 of 9 falls occurred 6–8pm” (range-aware)
- **Where**: “Memory Care” (if applicable)
- **Why it matters**: short risk framing (1 line)
- **CTA**: “View evidence” (deep link to filtered incidents list)

Allowed insight categories (v1):
- time-of-day clustering (shift windows)
- unit clustering
- repeat residents clustering
- documentation bottleneck clustering (missing fields)

---

## Guardrails (Critical)

- Never output an insight without:
  - a quantitative evidence line
  - a drilldown link to the exact evidence set
- Avoid causal claims; use observational language (“correlated”, “clustered”).

## Success Criteria

- [ ] Card renders in E5 position.
